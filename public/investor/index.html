Valuation Methodology

SAFE + Token Warrant positions (Hyperbeat, Rise, Resolv, Symbiotic): cash deployed purchases equity via a SAFE; a separate token warrant grants a token allocation exercisable at nominal cost. Equity is held at cost basis; the token allocation is marked at current market where the token is live, and added to the equity cost to produce the Position Mark.

SAFT positions (ETHGas Investment): cash deployed purchases tokens directly. Position Mark = tokens × current price.

Strategic allocations (Hyperbeat Strategic, ETHGas Strategic): tokens granted with no cash cost basis. Position Mark = tokens × current price where live; no cost basis means multiples are not meaningful.

Pre-TGE positions: held at cash cost basis. Multiple shown as 1.00x pending tokenization.

Vesting: computed from each position's contractual schedule against today's date. Charts show cumulative percent unlocked from TGE/effective date through full unlock.

Prices: fetched live from CoinGecko on page load. If the API is unavailable, last-known prices are used.
